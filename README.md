# 我的第一个免费网页

这是一个可以直接部署到 GitHub Pages 的静态网页项目。

## 文件说明

- `index.html`：网页内容
- `styles.css`：网页样式

## 免费部署方式

1. 在 GitHub 新建一个仓库，例如 `free-web-demo`
2. 上传本文件夹里的 `index.html`、`styles.css`、`README.md`
3. 进入仓库的 `Settings`
4. 左侧点击 `Pages`
5. `Build and deployment` 选择：
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
6. 点击 `Save`

几分钟后，GitHub 会生成一个类似下面的网址：

```text
https://你的GitHub用户名.github.io/free-web-demo/
```
